## Integration

1- Add it in your root build.gradle at the end of repositories:

<pre>
allprojects {
    repositories {
        ...
        <b>maven { url 'https://jitpack.io' }</b>
    }
}
</pre>

2- Add the dependency

<pre>
dependencies {
    ...
    <b>implementation 'com.github.MohamedWael:UtilsLibrary:1.0.2'</b>
}
</pre>
